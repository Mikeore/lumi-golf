# Current Evidence from LUMI

This repository is derived from ideas developed in my broader LUMI research project.

At the current stage, the strongest small-scale signals motivating this Parameter Golf branch are:

- reliable grokking-style success on modular arithmetic across multiple seeds
- a formal structural holdout benchmark where the architecture consistently outperforms a baseline on balanced accuracy
- evidence that compact, structure-biased modeling can produce useful generalization gains under constrained settings

These results are not direct Parameter Golf results. They are the motivation for creating a dedicated compact language-model branch tailored to the challenge.
